## ✦ Ключевые возможности

- **Полная невидимость** — окно исключено из захвата экрана, не видно в Zoom, Google Meet, Teams и других трансляциях
- **Распознавание речи** — установка whisper.cpp в один клик (CPU или GPU/CUDA), либо свой whisper-сервер
- **Чат с LLM** — любой OpenAI-совместимый эндпоинт (DeepSeek, OpenRouter, локальный Ollama)
- **Vision** — вставка скриншотов в чат (`Ctrl+V`), захват и анализ экрана через отдельную vision-модель
- **Карточки** — сохранение диалогов в карточки с автосохранением во время общения
- **Локальные модели** — работа без интернета для приватности и скорости
- **Кроссплатформенность** — Windows, macOS, Linux
- **Русский интерфейс**

## ▸ Как использовать

### 1. Настройка

1. Откройте KnowGhost
2. Перейдите в `Settings` → `AI`
3. Введите **API ключ** (OpenAI, Google AI, или любой OpenAI-совместимый провайдер)
4. Выберите **LLM модель**
5. Выберите **источник аудио**:
   - `System Audio` — захват звука из приложений (Zoom, Teams и т.д.)
   - `Microphone` — запись с микрофона
6. Настройте **модель транскрипции** и промпт

### 2. Использование

1. Переключитесь на вкладку `Main`
2. Нажмите `Start Audio Loop` — запись в фоне
3. Когда нужно, нажмите горячую клавишу или `Send Last X Seconds` для получения ответа от AI
4. Получайте мгновенные ответы во время собеседований и экзаменов

### 3. Советы

- Используйте **системный звук** для захвата вопросов интервьюера
- Настройте **прозрачность** и **поверх всех окон** для скрытности
- Настройте **горячие клавиши** для быстрых действий
- Тренируйтесь перед важными событиями

## ⊕ Локальная работа

KnowGhost поддерживает полностью локальную работу — без облачных сервисов, максимальная приватность.

### Локальный LLM

1. В настройках: `Mode → LLM` = `Local`
2. Выберите модель (например, `qwen3:4b` для слабых ПК)
3. Установите [Ollama](https://ollama.com/)
4. `ollama pull qwen3:4b` затем `ollama serve`

### Локальное распознавание речи

1. В настройках: `Mode → Transcription` = `Local`
2. Выберите `Source` = `Managed` (установка в один клик) или `External` (свой whisper-server)
3. Для managed: выберите вариант (`CPU` или `GPU/CUDA`), нажмите `Install`
4. Приложение автоматически скачает и настроит whisper.cpp
